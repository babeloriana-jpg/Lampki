# Projekt: Landing Page Khordong - Program 2026

## Cel projektu
Stworzenie strony typu landing page prezentującej wydarzenie (Kurs Podarowania Lampek Maślanych) w roku 2026 w Ośrodku Dropan Ling (Buddyzm Tybetański Linii Dziangter i Khordong). Strona ma budować atmosferę spokoju, profesjonalizmu i zapraszać do praktyki buddyjskiej.
**Model budowy strony:** AIDA (Attention, Interest, Desire, Action).

## Struktura zasobów (Ważne dla agentów)
- `_materialy/zrodlo.md` – Główny wsad merytoryczny, opis kursu i wskazówki co do wyglądu strony
- `_materialy/Images/` – Tutaj znajdują się wszystkie zdjęcia do wykorzystania na stronie ( do wyboru).
- `_materialy/klaudiusz_i_czarek.md` – Wasz wspólny plik do rozmów i ustaleń.

## Wytyczne techniczne
- Budujemy landing page w czystym HTML/CSS/JS.
- Strona musi być responsywna (Mobile First).
- Wdrożenie docelowe: Netlify.
- Klaudiusz: Pamiętaj, aby ścieżki do zdjęć w kodzie zawsze wskazywały na `_materialy/Images/nazwa_pliku.jpg`.
## Wdrożenie: 
- Repozytorium GitHub połączone z Netlify (Continuous Deployment).

Standardy Gita: Klaudiusz dba o deskryptywne komunikaty commitów (zgodnie ze swoją tożsamością).

Struktura: Plik index.html musi znajdować się w folderze głównym (root), aby Netlify poprawnie go wykryło.

## Role
- **Czarek:** Odpowiada za strukturę (AIDA), psychologię przekazu i ostateczne teksty.
- **Klaudiusz:** Odpowiada za perfekcyjne kodowanie, UX i efekt "WOW".
